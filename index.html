Our Mission

Our goal to teach people how to code for free, with using YHWH-centered projects. Coding is a skill that can help you find a flexible, remote career, allowing you to spend more time seeking YHWH, and less time working dead-end jobs.

Why choose us?

We offer free, easy-to-follow tutorials that will teach you how to code in as little as 8 weeks. Plus, we have more resources on our resources page that can help you on your coding journey.

How does it work?

It's super easy. Watch the course videos, code along with us and complete the assigned activities. Before you know it, you'll have the skills necessary to become a professional front-end web developer.
